# Тестовое задание — Бизнес-аналитик / Project-менеджер

Репозиторий создан для передачи выполненного тестового задания по проекту **«AI-агент проверки целевого использования льготных кредитов»** (Сбер, заказчик — Е. С. Сафронова, руководитель проектов).

## Структура репозитория

```
ReqForApproval_AI_Agent/
└── src/
    └── AI-агент льготные кредиты — тестовое задание.pdf   ← выполненное тестовое задание
```

Файл `src/*.pdf` — итоговый документ с ответом на тестовое задание в требуемом формате (PDF).

## Что внутри PDF

**Часть I. Тестовое задание**
1. 10 вопросов к заказчику и стейкхолдерам (казначейство, бухгалтерия, юристы, IT-департамент) — с указанием, кому адресован вопрос, зачем задаётся и какой риск закрывает
2. Ключевой бизнес-процесс «Первичная проверка пакета документов перед авансовым платежом» — AS IS / TO BE + схема процесса
3. Дорожная карта пилота на 2 месяца (7 вех, критерии готовности, вовлечённые роли, ключевые риски)
4. Процесс работы с заказчиком и командой разработки (формат, частота, фокус коммуникации)

**Часть II. Мотивация участия в проекте**
- Почему интересен проект
- Роль в команде
- Время и период участия
